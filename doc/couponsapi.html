/coupons get post

get

list all the coupons (admin only)

post

create a new coupon. signedToken is used for admin to create coupon with rules. Post request without signedKey will create coupon with default rule regardless of the rules passed in, ie. ignoring couponRule and rebateRule.

get /coupons

list all the coupons (admin only)

  • Request

Query Parameters

  • signedToken: required (string)

post /coupons

create a new coupon. signedToken is used for admin to create coupon with rules. Post request without signedKey will create coupon with default rule regardless of the rules passed in, ie. ignoring couponRule and rebateRule.

  • Request
  • Response

Body

Type: application/json

Schema:

coupon without coupon code

Example:

{
  "username": "test1",
  "mobile": "12345356467",
  "couponRule" : {
    "type" : "perc",
    "value" : 10
  },
  "rebateRule": {
    "type": "cash",
    "value": 100
  },
  "signedToken" : "dfsdoiuwqe"
}

HTTP status code 201

Body

Type: application/json

Schema:

{
   "$schema": "http://json-schema.org/draft-04/schema#",
   "type": "object",
   "properties": {
      "couponCode" : {
        "type" : "string"
      },
      "username" : {
        "type": "string"
      },
      "couponRule" : {
        "type" : "object",
        "properties" : {
          "type" : {
            "type" : "string",
            "enum": ["perc", "cash"]
          },
          "value": {
            "type" : "number",
            "minimum": 0,
            "description" : "must be less than 100 when type is perc."
          }
        },
        "required" : ["type", "value"]
      },
      "rebateRule" : {
        "type" : "object",
        "properties" : {
          "type" : {
            "type" : "string",
            "enum": ["perc", "cash"]
          },
          "value": {
            "type" : "number",
            "minimum": 0,
            "description" : "must be less than 100 when type is perc."
          }
        },
        "required" : ["type", "value"]
      },
      "valid" : {
        "type" : "boolean",
        "defualt" : true
      }
   },
   "required": ["couponCode", "username", "couponRule", "rebateRule"]  
}

Example:

{
  "couponCode" : "test1perc10",
  "username": "test1",
  "couponRule": {
    "type" : "perc",
    "value" : 10
  },
  "rebateRule": {
    "type": "cash",
    "value": 100
  }
}

HTTP status code 406

returned when Json posted has invalid properties

Body

Type: application/json

Example:

{
  "message" : "mobile is required."
}

/coupons/{couponCode}/orders get post

get

list all the orders linked to this coupon, signedToken required.

post

link an order with this coupon, and return the discounted order for verification

get /coupons/{couponCode}/orders

list all the orders linked to this coupon, signedToken required.

  • Request

URI Parameters

  • couponCode: required (string)

Query Parameters

  • rebated: required (boolean - default: false)
  • showTotalNum: (boolean - default: false)

    only the total number of orders lined to this coupon code.

  • signedToken: required (string)

post /coupons/{couponCode}/orders

link an order with this coupon, and return the discounted order for verification

  • Response
